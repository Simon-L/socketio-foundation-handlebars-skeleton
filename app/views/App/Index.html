Socket.io + Foundation + Handlebars skeleton

WebSockets, CSS/JS framework and client-side templating

This is the default index page.

You should see a Foundation success alert box generated from Handlebars templates if everything worked.

Check Index.html in app/views/App/, main.js in public/js/ and the console output for more infos.

What's inside:

Socket.io


Simple and complete WebSocket connections.
Find specific handlers for receiving and sending messages in Go in app/handlers.go and general Socket.io server setup in app/socketio.go.
Server uses package go-socket.io.
Read more on Socket.IO

Foundation


UI Framework using JS and CSS, contains all you need to start a new app.
Read more

Handlebars


Rendering client-side templates compatible with Mustache.
You need Handlebars to precompile templates.
Run handlebars . -f ../../../public/js/handlebars-templates.js in app/views/handlerbars to precompile your templates and find them as Handlebars.templates.templatename() in Javascript.
Read more
